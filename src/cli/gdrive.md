# Gdrive

- [go fork1](https://github.com/carstentrink/gdrive)

```bash
git clone --depth=1 https://github.com/carstentrink/gdrive
cd gdrive
./compile
cp ./bin/gdrive* ~/go/bin/gdrive
```
